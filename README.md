## 問題内容
与えられる数値はツタの本数とそれぞれのツタの長さだけ <br>
これらの値を使用してそれを絵として表現してください。 <br>
条件は下記の通り <br>
・一番上の行と一番下の行には+------と枠を書く。幅は本数によって伸びる <br>
・2行目以降は一番左に"|"、偶数列には".",　１以外の奇数列にはツタの長さによってXもしくはVで表現する。ツタの末端はV。 <br>
<br>
例：ツタの本数 = 3、　それぞれのツタの長さ = 3, 1, 5 <br>
<br>
回答：
```
+------- 
|.X.V.X. 
|.X...X. 
|.V...X. 
|.....V. 
+-------

```
